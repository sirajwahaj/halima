# 🧁 Halima's Cakes - Online Ordering

A stunning, modern online cake ordering website built for GitHub Pages. Features an interactive menu, shopping cart, order form, and beautiful animations.

## ✨ Features

- **Full Product Menu** - 12+ cake varieties across Classic, Chocolate, Fruit, Specialty, and Vegan categories
- **Cake Pops Collection** - 8 unique cake pop flavors with bundle deals
- **Custom Orders** - Tiered pricing system (Simple, Themed, Premium)
- **Shopping Cart** - Add/remove items, quantity control, persistent via localStorage
- **Order Form** - Complete order submission with delivery options
- **Responsive Design** - Beautiful on all devices (mobile, tablet, desktop)
- **Smooth Animations** - Scroll-triggered animations, hover effects, parallax particles
- **Testimonials Slider** - Auto-playing customer reviews
- **Gallery Section** - Showcase of creations with hover overlays
- **Accessibility** - Reduced motion support, semantic HTML

## 🚀 Live Demo

Visit: `https://sirajwahaj.github.io/halima/`

## 🛠️ Tech Stack

- Pure HTML5, CSS3, JavaScript (No frameworks/dependencies)
- CSS Custom Properties for theming
- Intersection Observer API for scroll animations
- LocalStorage for cart persistence
- Google Fonts (Playfair Display, Poppins, Dancing Script)
